## EMERGENCY PROCEDURES

### IF POSTING DELAYED (0-15 minutes):
- Post as soon as possible
- Add note: "Apologies for the delay - wanted to make sure this important insight was shared!"
- Proceed with normal engagement

### IF POSTING FAILED (15+ minutes delay):
- Assess technical issue
- Try LinkedIn mobile app as backup
- If still failing, save as draft and post next available slot
- Update followers: "Technical issues today - will share this important post tomorrow at [time]"

### IF LOW INITIAL ENGAGEMENT:
- Boost by sharing in 3-5 relevant groups
- Tag additional relevant connections
- Consider $10-20 LinkedIn boost if strategic
- Engage with similar posts to increase visibility

### IF NEGATIVE COMMENTS:
- Respond professionally and empathetically
- Address concerns directly
- Offer to continue conversation privately
- Never delete negative comments (unless spam/harassment)
