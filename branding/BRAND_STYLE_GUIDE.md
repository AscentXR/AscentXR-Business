# Ascent XR Brand Style Guide

## Overview
Ascent XR is an AI-powered spatial learning platform that transforms education through immersive experiences. Our brand combines the innovation of XR technology with the approachability of modern educational tools.

## Color Palette

### Primary Colors
- **Ascent Blue**: `#2563EB` - Trust, intelligence, innovation
- **Learning Purple**: `#7C3AED` - Creativity, wisdom, imagination

### Secondary Colors
- **Success Green**: `#10B981` - Growth, achievement, safety
- **Energy Orange**: `#F59E0B` - Enthusiasm, engagement, warmth
- **Neutral Gray**: `#6B7280` - Balance, stability, professionalism

### Accent Colors
- **Sky Blue**: `#0EA5E9` - Clarity, openness
- **Lavender**: `#A78BFA` - Creativity, calmness
- **Coral**: `#FB7185` - Energy, attention
- **Mint**: `#34D399` - Freshness, growth

### Usage Guidelines
- **Primary**: Blue for primary CTAs, Purple for secondary actions
- **Secondary**: Green for success states, Orange for highlights
- **Accents**: Use sparingly for emphasis and visual interest

## Typography

### Font System
- **Headlines**: Inter Bold (`font-weight: 700`)
- **Subheadlines**: Inter SemiBold (`font-weight: 600`)
- **Body Text**: Inter Regular (`font-weight: 400`)
- **UI Text**: Inter Medium (`font-weight: 500`)
- **Monospace**: JetBrains Mono (for code/technical elements)

### Scale
- **H1**: 48px (3rem)
- **H2**: 36px (2.25rem)
- **H3**: 30px (1.875rem)
- **H4**: 24px (1.5rem)
- **H5**: 20px (1.25rem)
- **Body**: 16px (1rem)
- **Small**: 14px (0.875rem)
- **Caption**: 12px (0.75rem)

### Line Heights
- Headlines: 1.2
- Body: 1.5
- UI Elements: 1.4

## Visual Style

### Spacing System
Based on 8px grid:
- **XS**: 4px
- **S**: 8px
- **M**: 16px
- **L**: 24px
- **XL**: 32px
- **XXL**: 48px
- **XXXL**: 64px

### Shadows
- **Small**: `0 1px 2px 0 rgba(0, 0, 0, 0.05)`
- **Medium**: `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`
- **Large**: `0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)`
- **Elevated**: `0 20px 25px -5px rgba(0, 0, 0, 0.1), 0 10px 10px -5px rgba(0, 0, 0, 0.04)`

### Borders & Radius
- **Small Radius**: 4px
- **Medium Radius**: 8px
- **Large Radius**: 16px
- **Pill**: 9999px
- **Border Width**: 1px or 2px

### Icons
- **Style**: Line icons with rounded corners
- **Size**: 16px, 20px, 24px, 32px
- **Color**: Primary or secondary palette
- **Stroke**: 1.5-2px weight

## AscentXR.com Design Principles (Analysis)

### Color Usage
- Dominant blues and purples create trustworthy yet creative atmosphere
- Clean whitespace with subtle gradients
- High contrast for readability
- Accent colors used sparingly for CTAs

### Typography Hierarchy
- Clear heading structure
- Sans-serif fonts for modern feel
- Consistent font weights throughout
- Good line spacing for readability

### Visual Weight & Spacing
- Generous padding around elements
- Clear visual hierarchy with size and color
- Balanced composition with breathing room
- Card-based layout with subtle shadows

### UI Patterns
- Clean card components
- Gradient backgrounds for hero sections
- Icon + text combinations
- Progress indicators
- Interactive elements with hover states

## Application Guidelines

### Digital Platforms
- **Web**: Use CSS variables system
- **Mobile**: Adapt spacing for smaller screens
- **Print**: Use CMYK color values

### Accessibility
- Minimum AA contrast ratio
- Support for keyboard navigation
- Clear focus states
- Alt text for all images

### Voice & Tone
- Professional yet approachable
- Clear and concise
- Educational but not academic
- Innovative but not confusing
